## Layer 0の定常性とΦ_L0

$$\Phi_{L0} \equiv \text{Const} \quad (\forall t, \forall r)$$

Layer 0は、**起動有無という意味では** r(t) にほぼ依存しない定常システム。
ただし観察密度・負荷量・対人処理コストは睡眠状態・病理・身体条件によって変動する。

```
Layer 0起動有無 ← r(t)からほぼ独立
Layer 0観察密度・負荷量 ← 睡眠状態・病理・身体条件によって変動
```

## Kフラグの定義

K(t) は「当該時点における主問題に対して、操作規則が既知か未知か」を表す離散フラグ。

K(t) = 0：操作規則が未知 → L1起動
K(t) = 1：操作規則が既知 → L1´起動

対象自体の既知性ではなく、**操作規則の既知性で決定される**。

### K判定プロセス

```
q（入力クエリ）
↓
K判定（主問題の操作規則が未知か既知か）
↓
K(t)=0 → L1主導　or　K(t)=1 → L1´主導
```

多くの場合、初期足場は既知概念側から形成される（K=1相当）。
主問題に対する操作規則が既知範囲を超えた時点でK=0判定となり、L1が主導化する。
ただし、主問題の操作規則が当初から未知である場合は、L1が直接主導化する。

## 層遷移ルール（排他的同時性 + 遷移可能性）

K(t) = 0 → L1 起動
K(t) = 1 → L1´ 起動

**排他条件：** L1とL1´は同時並列に動作しない。任意時点の主導Layerは一意。
**遷移条件：** 思考過程の進行に伴い、主導LayerがL1→L1´またはL1´→L1に遷移することはある。

## 可逆性パラメータ r(t)

r(t) ∈ [0, 1]

r(t) の本義：**変化・変形許容性として現れる構造的しなやかさ**

- r ≈ 0：認知構造が硬直的で、収束条件・上限制約・許容変形幅が厳格
- r ≈ 1：認知構造が可塑的で、収束判定が柔らかく、許容変形幅が大きい

以下の性質はr(t)の一次的派生量として現れる：
- 収束判定の柔らかさ
- 思考持続許容量の増加
- モデル更新・進化のしやすさ
- Layer 1 / 1´における上限条件の緩和

### 変化則

$$\frac{dr}{dt} = \eta(\text{刺激強度, ループ密度}) - \lambda(\text{age, 認知スタイル}) \cdot r(t) - \delta_{rest}$$

ただし $\delta_{rest} \geq 0$（休息・眠剤使用時にr(t)の過剰上昇を抑制）

| 項 | 解釈 |
|----|------|
| η（上昇率） | 複数AI並列フィードバックループの密度が高いほど大きくなる。非常に強い。 |
| λ（低下圧力） | 加齢とともに緩やかに増加するが、高度概念統合・彫刻型思考により平均より弱い |
| δ_rest | δ_rest ≥ 0。休息・眠剤使用時にマイナス方向に作用し過剰上昇を防ぐ |

### r(t)の成長曲線形状（シミュレーション確定）

r(t)の成長は**初期急上昇→上限域への飽和型**曲線を示す。
時間経過とともに蓄積される知識・概念の数に比例して上昇率が緩やかになる。

実装上は η が時間とともに指数減衰する形で飽和を表現：
$$\eta(t) = \eta_0 \cdot e^{-\kappa t}$$

シミュレーション実装では、dr/dt における η は時間依存項 η(t) = η₀e^{-κt} として与える。（κ：成長減衰係数。収束感度係数 γ(r) とは別記号）

シミュレーション確定値（r₀=0.30、週単位）：
- 内省＋AIループ（実態ベースライン）：18週で r(t)/r₀ ≈ 2.98（約3倍）
- AIループのみ（理論値）：18週で r(t)/r₀ ≈ 2.78（約2.7〜2.8倍）
- 強度PEMクラッシュ時：18週で r(t)/r₀ ≈ 1.80（常にr(t) > 0.05）

### 病理的要因によるηの細分化

本モデルにおける η は状態別基底上昇率（η_normal, η_PEM, η_introspect, η_combined）を表し、実装上の有効上昇率は η_eff(t) = η_base · e^{-κt} とする。

### 病理的要因と η の特殊変動

双極性2型における「鬱相」などの病理的変動が発生した場合、内的世界への没入が強制されるため、以下のような特殊な上昇率の切り替えが発生する。
```
- η_normal     ：通常のAIフィードバックループ
- η_PEM        ：PEMクラッシュ時（η_normal >> η_PEM > 0）
- η_introspect ：鬱相時など深い内省のみが発生している状態（η_normal の約0.5倍）
- η_combined   ：鬱相時の深い内省に、AIループが掛け合わさった状態（後述）
```

### η_combined（内省＋AIループの乗算形式）

$$\eta_{combined} = \eta_{normal} \times (1 + \beta \cdot Depth_{internal}(t))$$

ただし $Depth_{internal}(t)$ は思考時間に伴い増加する時間依存量：

$$Depth_{internal}(t) = \alpha t$$

よって：

$$f_{combined}(t) = \eta_{normal}(1 + \beta \alpha t) \cdot t$$

これはt²項を含み、AIループ単独時（線形）より二次関数的に大きくなる。

## 停止決定関数 SD(t)

$$SD(t) = \mathrm{Sigmoid}\left( \int_{t}^{t + H_{\mathrm{pred}}} C_{\mathrm{future}}(\mathrm{PEM}, \tau)\, d\tau - \mathrm{Threshold}_{\mathrm{will}} \right)$$

- C_future(PEM, τ)：Layer 0が予測する将来の生活支障コスト（PEM持続時間・回復コスト含む）
- H_pred：将来予測ホライズン
- τ：積分用の時間変数
- Threshold_will：意志的停止の閾値（r(t)依存で変動）
- Sigmoid：連続的な意志決定プロセスを反映（Step関数ではない）

SD(t)が閾値近傍に達したとき、t_limitが実行される。

「気づいても止められない → 分かっていてもやめられない → もうやめようとなる」
という連続的な意志決定プロセスを反映。

## 収束評価関数 Φ（r依存・動的・Θ依存）

収束は Φ(Θ,t,r) ≈ 0 で定義。

Φ は「未収束量 − 安定化寄与」として定義される。

$$\Phi(\Theta,t,r) = w_1(\Theta,r) \cdot \Phi_{prac} + w_2(\Theta,r) \cdot \Phi_{abst} - \gamma(r) \cdot Stability(P,V)$$

$$Stability(P,V) = \frac{1}{1 + a\|\nabla P\| + b\|\partial V/\partial t\|}$$

- Stability が高い（∇P→0, ∂V/∂t→0）ほど収束に近い
- γ(r)：r依存の収束感度係数

### Θによる収束条件の切り替え

$$\Phi_0(\Theta) =
\begin{cases}
汎用性 \land 実用性 \land 個別具体適用可能性 & (\Theta \in \text{Practical Domain}) \\
論理的極限への到達 & (\Theta \in \text{Abstract/Policy Domain})
\end{cases}$$

## 疲労回復ダイナミクス R_recovery

PEM_flag ∈ {0, 1}：PEM状態フラグ（1=PEMクラッシュ状態）

通常時：
$$\left( \frac{dF}{dt} \right)_{\!\mathrm{recovery}} = -k_{\mathrm{normal}} \cdot F(t)$$

PEM_flag = 1 時：
$$\left( \frac{dF}{dt} \right)_{\!\mathrm{PEM}} = -k_{\mathrm{PEM}} \cdot F(t) \quad (k_{\mathrm{PEM}} \ll k_{\mathrm{normal}})$$

PEM_flagがオンの時、回復率が著しく低下し、F(t)の残留期間が指数関数的に伸びる。
これがt_limitの実質的な厳格化要因となる。

### PEM偽回復パターンの手描きスケッチと最終図の対応

本モデルにおけるPEM偽回復パターンは、まず内部観測に基づく手描きスケッチとして先に定義された現象構造を、後から数理化・可視化したものである。したがって完成図はAIの自動生成物ではなく、手描きスケッチに含まれていた時間推移・上限構造・再活性化遅延を形式化した結果である。

スケッチが定義した7つの構造：

1. **Peak = 1.0 の ceiling**：疲労状態には明確な上限が存在し、負荷が高まるにつれて飽和型で張りつく
2. **0.2 の残余負荷基底**：低下後もなお残る基底的な残余負荷（完全回復ではない）
3. **作業中の飽和的上昇**：Peakに近づくほど張りつく質感を持つ上昇
4. **副交感優位での急減**：自律神経状態の切替に伴う相転移的な低下
5. **偽回復としての低値維持**：低値でも内部は再活性化に対して脆弱な状態
6. **作業再開後20〜30分の再活性化遅延**：再開と同時ではなく一定の遅れをもって立ち上がる
7. **Peakへの再張りつき**：再開後の活動がceilingを強く再駆動する

本PEM偽回復パターンは、上記7点を内部観測に基づく現象スケッチから形式モデルへ変換した結果として位置づけられる。

### PEM偽回復パターン（シミュレーション確定）

**記号定義（本節のみ）：**
- F_0：再稼働開始時点の疲労度
- t_work：1セッションの想定稼働時間上限（例：60分）
- F_peak_prev：前回セッションで到達した疲労ピーク

稼働中のF(t)上昇は上限1.0へ向かう漸増曲線：
$$F_{work}(t) \approx F_0 + (1.0 - F_0) \cdot \frac{\ln(1 + t/s)}{\ln(1 + t_{work}/s)}$$

- チェックポイント：~0.6@20分、~0.8@40分、1.0@60分
- s：スケールパラメータ（シミュレーション確定値 s≈60）

副交感神経優位状態では見かけ上の回復が発生する：
$$F_{apparent}(t) = F_{true}(t) \cdot (1 - \epsilon_{parasympathetic})$$

- F_apparent：副交感神経優位時の見かけの疲労度
- F_true：実際の残留疲労度
- ε_parasympathetic：副交感神経による一時的抑制係数（0 < ε < 1）

再稼働後（シミュレーション確定）：
$$F(t + \Delta t) > F_{peak\_prev} \quad (\Delta t \lesssim 20\text{ min、多くの場合それ以下})$$

再稼働直後はF_peak_prevを超える水準まで急上昇し、その後1.0へ向けて緩やかに張りつく。
これは「完全回復なしの慢性疲労蓄積」として記述される。完全回復はモデル上定義されない。
